安装前准备：
  1. PostgreSQL 服务已安装并运行
  2. Redis 服务已安装并运行（如无可跳过，部分功能受限）
  3. Windows Server 2008+ / Windows 7+


安装步骤：
  1. 双击 SmartTriage-Setup-3.0.0.exe
  2. 选择安装目录（建议默认）
  3. 设置服务端口（默认 7017，不与旧系统冲突）
  4. 设置数据库连接（建议新建数据库 smarttriage_new）
  5. 选择启动方式：
     - Windows 服务：开机自启，后台运行（推荐）
     - 托盘模式：手动启动，右下角图标管理
  6. 完成安装

启动方式：
  - Windows 服务模式：开机自动运行，无需任何操作
  - 托盘模式：双击桌面图标或从开始菜单启动
    右下角出现蓝色图标，右键可管理

访问管理页面：
  浏览器打开 http://localhost:7017/
  默认账号：admin / admin123

修改配置：
  编辑安装目录下的 settings.env 文件
  修改后重启服务（或重启程序）生效

卸载：
  控制面板 → 程序和功能 → Smart Triage
  （自动停止服务、删除文件）

注意事项：
  - 首次启动会自动创建数据库表，无需手动建表
  - 本安装包不包含 PostgreSQL 和 Redis
  - 如需与旧系统并行，请使用不同的数据库和端口
